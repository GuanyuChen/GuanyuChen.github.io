CSS预处理器-less篇

简介

Less 是一门 CSS 预处理语言，它扩展了 CSS 语言，增加了变量、Mixin、函数等特性，使 CSS 更易维护和扩展。

Less 可以运行在 Node 或浏览器端。

使用方法

安装

$ npm install -g less

命令行使用

将less包全局安装，就可以命令行调用lessc命令来编译.less文件

$ lessc styles.less

将编译结果输出到指定的.css文件

$ lessc styles.less styles.css

使用clean-css插件将编译结果压缩后输出到指定文件

$ lessc --clean-css styles.less styles.min.css

lessc命令查询帮助

$ lessc

浏览器端使用

引入less.js文件建议在开发环境中使用，不建议在生产环境中使用

将link标签的rel属性设置为rel = "stylesheet/less"

<link rel="stylesheet" type="text/css" href="styles.less" />

引入less.js文件

<script src="less.js" type="text/javascript"></script>
注意：
  1. 样式表要在less.js前引入
  2. 由于每个.less文件是单独编译，所以内部定义的变量，混合类型和命名空间不能访问
  3. 由于浏览器加载外部资源的同源策略，需要启用CORS
选项

React 概览

核心思想 : 封装组件，各个组件维护自己的状态和UI，当状态变更，自动重新渲染整个组件。

基于这种方式的一个直观感受就是不再需要不厌其烦地来回查找某个DOM元素，然后操作DOM去更改UI。

核心概念 :

ES6基本语法

1.Let + Const 块级作用域和常量

let

  1. 定义局部变量，只在当前作用域内有效
  2. 不存在变量提升
  3. 不受外界作用域影响
  4. 不允许重复声明

const

  1. 定义常量，只在当前作用域内有效
  2. 声明后，常量值不能改变
  3. 不存在变量提升
  4. 不能重复声明
  5. 指向常量所在地址
