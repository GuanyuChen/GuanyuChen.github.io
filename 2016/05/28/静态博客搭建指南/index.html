静态博客搭建指南

发表于 2016-05-28   |   分类于 笔记 // 插入位置

利用Github搭建技术博客

环境准备

  • 安装git
  • 安装Node.js
  • 安装hexo
    $ npm install -g hexo

hexo是一款基于Node.js的静态博客框架

创建Hexo文件夹

在想要的路径创建一个Hexo文件夹，然后git bash 进入该路径

hexo init

Hexo会自动在目标文件夹下建立网站需要的所有文件

hexo使用

常用命令

hexo generate 生成静态页面，可以在public目录查看整个网站的文件

hexo server 启动本地服务，进行文章预览调试(http://localhost:4000) ctrl+c关闭

hexo clean 清除缓存 **强烈建议每次执行命令前先清理缓存，每次部署前先删除.deploy**

hexo deploy 部署.deploy目录

hexo new "postName" 新建文章

在_posts目录下会生成文件标题.md

编辑完后保存，hexo server预览

hexo new page "pageName" 新建页面

hexo help 查看帮助

部署步骤

  1. hexo clean
  2. hexo generate
  3. hexo deploy

有以下提示说明部署成功

[info] Deploy done:git

目录结构

├── .deploy #需要部署的文件
├── node_modules #Hexo插件
├── public #生成的静态网页文件
├── scaffolds #模板
├── source #博客正文和其他源文件，404、favicon、CNAME 都应该放在这里
| ├── _drafts #草稿
| └── _posts #文章
├── themes #主题
├── _config.yml #全局配置文件
└── package.json
