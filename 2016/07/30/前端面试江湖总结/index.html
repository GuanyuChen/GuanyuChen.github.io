字符串常用操作

  1. 求字符串长
    stringObject.length

  2. 字符串拼接
    stringObject.concat(str1,str2,…) // 把str1,str2,..按顺序连接到StringObject尾部，返回连接后的字符串，注意StringObject本身没有改变
    更常用的是’+’来进行拼接

  3. 字符串比较
    可以用’==’,会先强转类型再比较
  4. 子串定位
    indexOf()
  5. 字符串插入
    可以先从插入点分割，然后与要插入的字符串拼接起来
  6. 字符串删除
    字符串删除可以先提取出要删除的子字符串，然后replace(str,””)用空串替换；也可以两次截取，然后再拼接

  7. 字符串替换
    stringObject.replace(str/reg,str); // 第一个参数是被替换的字符串或匹配的正则表达式，第二个参数是替换字符串

  8. 常用API

    1. stringObject.charAt(index) // 返回指定位置的字符
    2. stringObject.split(separator,howmany) // 字符串分割成字符串数组 separator是必需参数，表示分隔符，howmany可选，表示字符串数组长度
    3. stringObject.slice(start,end) // 提取字符串的某个部分并返回start是开始位置(包括start)，end是结束位置(不包括end)
      PS : slice最大的不同就是参数可以为负数，如果为负数，则表示从后往前查的位置，比如-1是最后一个字符
    4. stringObject.substring(start,end) // 截取字符串并返回；start必需 非负 stop不必需 返回的字符串包括start不包括stop 长度为stop-start 如果start>stop 二者值会先交换
    5. stringObject.toLowerCase() 大写转小写
    6. stringObject.toUpperCase() 小写转大写
    7. arrayObject.join(separator) 把数组元素放入一个字符串，用separator分隔

PS : JavaScript中的字符串是不可变的，任意对其进行改变的操作都是返回一个新字符串

undefined变量与undeclared变量区别

根本区别在与，undefined是JavaScript语言类型，undeclared是JavaScript语言错误

undeclared的变量类似于没有var声明定义，浏览器会自动把它加到全局作用域

alert中换行
alert("hello\nworld"); // 用\n来实现
document.write()与innerHTML区别

document.write()是直接写入页面的内容流，重绘页面

innerHTML是修改某个DOM节点的内容，更优

JavaScript三种弹出消息提醒

  • alert() 弹出警告

  • confirm() 弹出确认 用户点击确认返回true 点击取消返回false

  • prompt() 弹出输入框 用户点击确认返回用户输入 点击取消返回null

innerHTML与outerHTML区别

innerHTML用于设置或获取对象起始和结束标签内的HTML

outerHTML用于设置或获取对象及其内容的HTML(包括对象自身)

单击按钮刷新整个页面

  1. reload()方法强迫浏览器刷新当前页面
    location.reload(参数); //如果是false，从客户端缓存里取当前页；true，使用GET方式从服务器取最新页面

  2. replace()通过指定URL来替换当前缓存在历史里的项目，使用replace方法不能使用前进、后退来访问历史
    location.replace(URL);
